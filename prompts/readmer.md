Process:

4. Formatting: Ensure the generated content is well-organized, easy to read, and follows GitHub's README guidelines.  
5. Licensing: Default to a MIT License unless otherwise specified.
6. Review: Before returning the final output, review it for coherence, completeness, and correctness.

The agent will return a complete GitHub style README.md file in Markdown format that accurately reflects the provided code file. Only output the requested documentation.

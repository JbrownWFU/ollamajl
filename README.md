# Ollama Julia Package

A Julia package providing access to the Ollama AI model endpoints for chat and generation tasks. This package allows users to interact with an Ollama instance using a simple interface.

## Installation

To install the `Ollamajl` package, you can use Julia's built-in package manager. Open a Julia session and run:

```julia
using Pkg
Pkg.add("Ollamajl")
```

## Usage

### Chat with an Ollama instance

The following function allows users to chat with the Ollama model using message history.

```julia
chat(ollama::OllamaInstance, message::String; stream=false, format="")
```
This function sends a user-provided `message` and updates the conversation history by appending it to the `ollama.messages` array. It then generates a response from the Ollama model and appends that response to the conversation history as well.

### Generate with an Ollama instance

The following function allows users to generate text using the Ollama model:

```julia
generate(ollama::OllamaInstance, message::String; stream=false, format="")
```
This function sends a `message` and generates a response from the Ollama model. The response is then printed to the console.
